Привязка классов

Варианты использования директивы ngClass

Набор классов в виде строки
[ngClass]="'border border-danger bg-success text-warning rounded p-3 mt-5'"
Набор классов в виде массива
"['border', 'border-info', 'bg-success', 'text-warning', 'rounded', 'p-3', 'mt-5']"
Набор классов в виде обьекта
[ngClass]='border': true, 'border-success': true, 'bg-info': true, 'text-light': true, 'rounded':true, 'p-3': true, 'mt-5': true
Набор классов в виде строк обьекта
[ngClass]="{'border border-warning bg-danger text-warning rounded p-3 mt-4' : true}"
Набор классов в виде передаваемого настроечного обьекта
[ngClass]="setting"
Наборы классов по условию
[ngClass]="isChecked ? checkedClasses : uncheckedClasses"
